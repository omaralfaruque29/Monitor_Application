This monitor application is a web application developed in python flask environment.
This application provides following features
-Data forecasting
-Anomaly detection
-Visualization
Tools: python flask, pandas, D3.js, Rest api, influxdb client etc.

Following tools are needed to run this application
pip install Flask
pip install influxdb
pip install pandas
pip install numpy
pip install matplotlib
pip install scipy
pip install statsmodels
pip install passlib
pip install psycopg2
pip install Flask-HTTPAuth
pip install itsdangerous
